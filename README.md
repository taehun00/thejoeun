## 🐶😺 **Why Pawject?**

사료 선택은 생각보다 어렵습니다.  
복잡한 라벨, 다양한 브랜드, 건강 상태에 따른 수많은 선택지…

> **“반려동물에게 정말 맞는 사료를 쉽게 고를 수 있다면?”**

이 질문에서 Pawject가 시작되었습니다.

우리는 **신뢰할 수 있는 데이터**, **직관적인 UI**, **기능 중심 설계**를 통해  
보호자의 선택을 돕는 편리한 웹 서비스를 만들고자 했습니다.

## 🛠️ **Tech Stack**

### **Backend**
- Java 11  
- Spring Framework 5.3 (MVC, Security)  
- MyBatis 3.5  
- Oracle Database 11g XE  

### **Frontend**
- HTML5 / CSS3 / JSP  
- JavaScript (ES11)  
- jQuery 3.7.1  
- Bootstrap 5  

### **Tools**
- Eclipse / IntelliJ  
- Git · GitHub  
- Figma (UI 설계)
