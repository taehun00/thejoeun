# 🐾 Pawject
**Paw + Project = 반려동물을 위한 개발 여정**

> 반려동물 보호자가 겪는 불편함에서 출발한 프로젝트입니다.  
> 사료 선택(라벨 비교/성분 확인)의 어려움과 건강·운동 정보 관리 니즈를 해결하기 위해  
> **사료 데이터 + 건강정보 + 커뮤니티 + 운동 기능**을 통합한 플랫폼을 구현했습니다.

---

## 🚀 Project Overview
- **프로젝트명**: 반려동물 건강 & 사료 종합 플랫폼
- **특징**: 하나의 프로젝트를 v1~v4로 나누어 **점진적 확장/고도화**
- **핵심 방향**: Prototype → Service Upgrade → API Expansion → React Architecture

## 🛠 Tech Stack (Summary)
- **Backend**: Java 11, Spring Boot, Spring Security, MyBatis / JPA
- **Database**: Oracle 11g, Redis
- **Frontend**: JSP, Thymeleaf → React(Ant Design)
- **DevOps**: AWS(EC2/S3/RDS), GitHub Actions CI/CD
- **External API**: OCR, ChatGPT, 채널톡, 날씨/지도 API

---

## 📌 Core Features (전체 버전 통합 기준)
- 멤버/펫 정보 관리 + 알림(웹소켓)
- 사료 데이터 관리(관리자) + OCR 기반 자동 입력
- 리뷰 커뮤니티: 다중 이미지 업로드 + ChatGPT 리뷰 순화
- 조건 기반 사료 검색 + 리뷰 연동 + ChatGPT 필터 추천
- 고객센터: FAQ + 1:1 문의 + 채널톡 실시간 상담
- 운동 게시판: 날씨/지도 기반 정보 + 운동 기능 연동
- 건강/질병 정보 제공 및 게시판 운영
- 이벤트: AI 기반 반려동물 캐릭터 생성
