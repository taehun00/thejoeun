# 🐾 Pawject v3
**반려동물 건강 & 사료 종합 플랫폼 (Service Expansion)**

> Paw + Project = 반려동물을 위한 개발 여정  
> **v2 서비스를 Spring Boot 기반으로 확장**하고,  
> OCR/ChatGPT/채널톡/날씨·지도 등 **외부 API 연동을 통해 운영 가능한 종합 서비스**로 고도화한 버전

---

## 🎯 Project Goal (v3)
**Pawject v3는 ‘기능 고도화’ 단계를 넘어, 외부 연동과 운영 흐름까지 갖춘 종합 플랫폼으로 확장한 버전입니다.**

- v2 서비스를 Spring Boot 기반으로 확장하여 **운영 가능한 서비스 구조**로 발전
- OCR/ChatGPT/채널톡/날씨·지도 API 등 **외부 API 연동**으로 사용자 편의성과 완성도 고도화
- 고객센터·사료 검색·리뷰·운동·건강정보 등 주요 기능을 추가하여 **종합 서비스로 확장**
- UI/UX 개선 및 데이터 흐름 최적화로 **사용자 동선/콘텐츠 접근성 강화**

## 🛠 Tech Stack (v3)
### Backend
- Java 11 (JDK 11)
- **Spring Boot 2.7 (MVC, Security)**
- **MyBatis 3.5**
- Thymeleaf
- RESTful API

### Database
- Oracle 11g

### Frontend
- HTML5 / CSS3
- Bootstrap 5
- jQuery
- AJAX

### Others / External
- JSON
- External API: **OCR, ChatGPT, 채널톡, 날씨/지도 API**

### DevOps / Collaboration
- Git / GitHub 기반 협업 및 버전 관리

---

## ✅ Main Features (v3)
> v3는 “게시판 중심 기능 + 검색/운영 + 외부 API 연동”을 결합해 플랫폼을 종합 서비스로 확장했습니다.

### ✅ 멤버 관리
- 회원/펫 정보 관리
- **웹소켓 기반 알림 기능**

### ✅ 사료 게시판 (관리자)
- 사료 데이터 등록/수정/삭제
- **OCR 기반 자동 입력(라벨 인식 → 입력 보조)**

### ✅ 리뷰 게시판
- 리뷰 작성/조회
- 다중 이미지 첨부
- **ChatGPT 기반 리뷰 문장 순화(문장 다듬기)**

### ✅ 사료 검색 기능
- 조건 기반 상세 필터 검색
- 검색 결과 → 리뷰 연동
- **ChatGPT 기반 필터 추천**

### ✅ 고객센터
- FAQ 기능
- 1:1 문의
- **채널톡 API 기반 실시간 상담 연동**

### ✅ 운동 게시판
- **날씨/지도 API 기반 정보 제공**
- 운동정보 게시판 ↔ 운동 스마트 게시판 연동

### ✅ 건강 정보 게시판
- 건강 정보 데이터 제공 및 관리 기능

### ✅ 이벤트
- **AI 기반 반려동물 캐릭터 생성 기능**

---

## ✨ What’s New (v2 → v3 개선점)
v3는 “기능 추가”보다 **외부 연동 + 운영 흐름 + 사용자 동선 강화**가 핵심입니다.

- 운영 가능한 서비스 구조로 확장(기능/데이터 흐름 정리)
- OCR/ChatGPT/채널톡/날씨·지도 API 연동으로 사용성 강화
- 사료 검색 기능 중심으로 콘텐츠(리뷰/게시판) 접근 동선 최적화
- UI/UX 개선을 통한 사용성 및 콘텐츠 탐색 효율 개선

---

## 🧱 Version Roadmap (v1~v4)
| Version | 핵심 목표 |
|---|---|
| v1 | 게시판 MVP 구현 + 데이터/화면 구조 설계 |
| v2 | 검색/페이징/AJAX/업로드 기반 실사용 서비스화 |
| **v3** | 외부 API 연동 + 고객센터/검색/운영 확장으로 종합 플랫폼화 |
| v4 | Spring Boot + React + JWT 기반 아키텍처 전환 |
